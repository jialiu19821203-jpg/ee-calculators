Theory

The non-inverting operational amplifier configuration uses negative feedback to control the voltage gain. The input signal is applied to the non-inverting (+) terminal, while the output is fed back to the inverting (-) terminal through the feedback resistor Rf.

The gain is determined by the ratio of the feedback resistor (Rf) to the ground resistor (R1). The gain is always greater than or equal to 1, and the output signal maintains the same phase as the input signal.

This configuration is widely used in analog circuits for signal amplification, buffering, and active filtering applications due to its high input impedance and precise gain control.
