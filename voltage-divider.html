Theory

A voltage divider is a simple passive linear circuit that produces an output voltage (Vout) that is a fraction of its input voltage (Vin). It is created by connecting two resistors in series.

The current flowing through the series circuit is consistent, allowing Ohm's Law to distribute the voltage drop across resistors R1 and R2 proportionally.

Detailed Theory & Application

The Mathematics

Vout = Vin √ó ( R2 / (R1 + R2) )

The division ratio is determined strictly by the resistance values. If R1 and R2 are equal, the output voltage is exactly half of the input voltage. This linear relationship makes it essential for signal scaling and reference generation.

Common Use Cases

  • Logic Level Shifting: Safe step-down of signals (e.g., 5V sensor output to 3.3V microcontroller input). Note: This is unidirectional and slow (unsuitable for high-speed I2C/SPI).
  • Sensor Interfacing: Reading resistive sensors like LDRs (Light Dependent Resistors) or Thermistors. The sensor acts as R2, and the voltage change is read by an ADC.
  • Bias References: Creating a mid-point voltage (e.g., Vcc/2) for biasing AC signals in single-supply Op-Amp circuits.

  • Loading Effect (Impedance Mismatch): This is the #1 error. If the device connected to Vout has low input impedance, it drags the voltage down. Rule of thumb: Load impedance should be >10x R2.
  • Power Dissipation: Ensure each resistor can handle the current flow (P = I¬≤R). Standard resistors are usually rated for 0.25W.
  • Not a Voltage Regulator: Never use a divider to power a load (like a motor or LED strip). The voltage will fluctuate wildly as the load current changes, and resistors will overheat.
